Bootstrap

Bootstrap merupakan sebuah kerangka kerja css(Css framework) open source untuk merancang sebuah situs web dan aplikasi web. Framework ini berisi template desain berbasis HTML dan CSS untuk tipografi formulir tombol dan navigasi.

Pencapaian

Pada materi ini diharapkan seluruh peserta memahami cara penggunaan framework bootstrap, dan bisa membuat sebuah tampilan landing page yang responsive dengan menggunakan bootstrap.

Syarat

Untuk pengambilan kelas bootstrap calon peserta harus Memahami html dan css dasar untuk lebih mudah memahami pada saat mulai pembelajaran.
